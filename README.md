# UFC Management System

<p align="center">


A standalone desktop application developed to centralize and automate the management of UFC fighter data, match scheduling, and performance analytics. This system features a custom **Pound-for-Pound (P4P) ranking algorithm** and secure role-based access.

## Key Features
*   **Role-Based Access Control:** Secure login for **Admins** (full control) and **Users** (view-only).
*   **Fighter Management:** Complete CRUD operations for physical stats and career history.
*   **Automated P4P Rankings:** Dynamic calculation based on win ratios, KOs, and performance metrics.
*   **Match & Event Scheduling:** Organize upcoming bouts with dedicated date and location management.
*   **Analytics Dashboard:** Visual insights into fighter distributions and system stats.

## Technical Stack
*   **Frontend:** Java Swing (GUI)
*   **Backend:** Java (JDK 8+)
*   **Database:** Oracle SQL Express Edition 21c
*   **Tools:** Git, Figma (UI Design)

## Database Schema
The system architecture relies on four primary tables:
1.  **Users:** Credential management and role assignment.
2.  **Fighters:** Physical attributes and career statistics.
3.  **Matches:** Individual bout details and outcomes.
4.  **Events:** UFC event scheduling and logistics.

## Getting Started

### Prerequisites
*   Java Development Kit (JDK) 8 or higher.
*   Oracle Database 21c installed and running.
